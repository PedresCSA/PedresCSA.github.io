# Personal Portfolio - Babylen Pedres

This is a simple personal portfolio website built using HTML and CSS with a sunset theme.

## Features
- Circular profile picture
- Sunset color theme
- Shield icons for skills
- Clean and modern design

## Files
- `index.html` – Main structure of the portfolio
- `style.css` – Styling and layout
- `profile.jpg` – Your profile image
- `README.md` – Project information

## How to Use
1. Put all files in one folder.
2. Add your image and name it `profile.jpg`.
3. Open `index.html` in any web browser.
